# QuestMatch Frontend

React + Vite + Tailwind CSS frontend for QuestMatch.

## Features
- **Signup/Login**: User authentication.
- **Discover**: Swipe through potential gaming matches.
- **Matches**: View mutual matches.
- **Chat**: Real-time (polled) chat with matches.
- **Profile**: Manage your bio, skill level, platforms, and games.

## Setup
```bash
npm install
```

## Running
```bash
npm run dev -- --host 0.0.0.0
```

## Tech Stack
- **React**: UI library.
- **Tailwind CSS**: Styling.
- **Framer Motion**: Animations for swiping.
- **Lucide React**: Icons.
- **Axios**: API communication.
- **React Router**: Routing.
